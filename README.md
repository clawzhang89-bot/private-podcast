# private-podcast

静态播客 feed 原型。

## 结构
- `feed.xml`：RSS feed
- `episodes/`：音频文件
- `site/cover.jpg`：封面（可选）

## 发布
后续可直接接 GitHub Pages。

## Demo status

- GitHub Pages 首页：`https://clawzhang89-bot.github.io/private-podcast/`
- RSS：`https://clawzhang89-bot.github.io/private-podcast/feed.xml`
- 首期案例音频：`episodes/2026-03-26-demo-analysis.mp3`
- 封面图：`site/cover.jpg`

iPhone 可在 Podcasts App 中通过 URL 订阅上面的 feed。
